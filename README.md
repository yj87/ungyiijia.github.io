# SECP1513
TECHNOLOGY AND INFORMATION SYSTEM Section 05

This course covers various aspects of IS/IT, including hardware, software, networking, and communications, will be discussed. Basic skills in PC installation and productivity tools will be gained through hands-on lab work, which will account for a significant portion of the study. Work-based learning includes industry visits and talks that will expose students to a real-world working environment, provide knowledge from the industry, and increase engagement between university and industry.
